This directory contains the scripts related to the analysis of graph properties (graph degrees).

#### Here is a brief description of the files:

[get_all_external_cluster_degrees.py](get_all_external_cluster_degrees.py) - Compute inter_cluster degrees, in-degrees, out-degrees and corresponding degree centralization values

[get_all_graph_degrees.py](get_all_graph_degrees.py) - Compute degrees, in-degrees, and out-degrees for the complete network

[get_all_internal_cluster_degrees.py](get_all_external_cluster_degrees.py) - Compute intra_cluster degrees, in-degrees, out-degrees and corresponding degree centralization values

[get_cited_articles_authors.py](get_cited_articles_authors.py) - Identify the most cited articles and their corresponding authors in a cluster

[get_graclus_cluster_scp_list.py](get_graclus_cluster_scp_list.py) - Get cluster-SCP list for Graclus clusters

[get_graph_citations.py](get_graph_citations.py) - Identify whether the (rated) cluster was cited in the cluster and/or in the graph

[get_in_cluster_author_citations.py](get_in_cluster_author_citations.py) - Compute the number of cited authors within the (rated) clusters and in the graph

[get_rated_author_95_citing.py](get_rated_author_95_citing.py) - Compute the number of citing authors who are at the 95th percentile of papers per author in the rated clusters
